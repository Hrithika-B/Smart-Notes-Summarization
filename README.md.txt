# 📝 Smart Notes Summarization

## 📌 Project Overview

Smart Notes Summarization is a Python-based application that automatically generates concise summaries from lengthy text or notes. It helps users quickly understand the key points of documents, making it useful for students, professionals, and researchers.

---

## 🚀 Features

* 📄 Automatic text summarization
* ✍️ Easy-to-use interface
* ⚡ Fast summary generation
* 🧠 Preserves important information
* 💻 Lightweight and simple to run

---

## 🛠️ Technologies Used

* Python
* Flask
* Natural Language Processing (NLP)
* NLTK
* Sumy
* HTML
* CSS

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Hrithika-B/Smart-Notes-Summarization.git
```

### 2. Move into the project folder

```bash
cd Smart-Notes-Summarization
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python app.py
```

---

## 📖 How to Use

1. Start the application.
2. Paste or type the text you want to summarize.
3. Click the **Summarize** button.
4. View the generated summary.
